[ToolTip] Info about a tweak: Move the cursor over a tweak to view a brief explanation. Tweaks marked with a *(asterisk) offer configuration options. To do this, use the provided "Options" button or right-click on entry. 
                                          
[Button] Check: Determines which tweaks are enabled and configured on your system or not. NO system changes are done yet! 
                                           
[Button] Apply: This will enable all selected tweaks.
                                           
[Button] Undo: This will restore the default Windows tweaks.
    
[Button] Load template: This will load a preset file with recommended tweaks.

[Button] Save template: This allows you to set your favorite tweaks and save them to a local file.
                                       
[State] Tweaks needs to be applied: This indicates that the Windows tweaks are in place.
                                           
[State] Tweaks already applied: This indicates that the recommended tweaks are in place.
